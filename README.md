# AHKgen

**A simple and intuitive GUI tool for generating AutoHotkey (AHK) scripts.**

AHKgen allows you to quickly create hotkeys, hotstrings, key remaps, and "send text" actions without writing AHK code manually.

---

## ✨ Features

- **Hotkeys** – Create combinations with modifiers (Ctrl, Shift, Alt, Win)
- **Hotstrings** – Auto-replace text with options:
  - Trigger instantly
  - Case sensitive
  - Trigger inside words
  - Raw text mode
- **Key Remapping** – Easily remap one key to another
- **Send Text** – Insert text (supports multiple lines and newlines)
- **Open URL** and **Run Command** actions
- **Import & Edit** existing AHKgen-generated scripts
- **Keyboard Layout Support** – QWERTY, QWERTZ, AZERTY
- **Dynamic macro counter** per category
- **Light/Dark Mode** – Automatically follows system settings
- **Multi-language Support** – EN, PL, ES, DE, FR, PT, CN, JP

### Technical Features
- Input validation and error prevention
- Script header indicating it was generated by AHKgen
- AHK version indicator in generated scripts
- Standard best practices (`#NoEnv`, `#SingleInstance`, `SendMode Input`, etc.)
- Lightweight thanks to Tauri + WebView2

## 🚀 How to Use

1. Run `AHKgen.exe`
2. Choose the action type (Hotkey, Hotstring, Remap, etc.)
3. Configure the keys or text
4. Add the action to your list
5. Click **Generate Script** and save the `.ahk` file
6. Run the generated script with AutoHotkey

> **Note**: The app will verify that loaded `.ahk` files were created by AHKgen.
